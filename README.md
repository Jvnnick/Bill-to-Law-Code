# Bill-to-Law-Code
Repository for the code used in the Master Thesis "From Bill to Law: Examining Gender Dynamics in Environmental Committees and Their Influence on Legislative Outcomes in Germany" by Julia Ebben

Code by Julia Ebben:
Code_for_Web_Scraper_for_Bavaria's_Documents.ipynb contains the code for scraping the Bavarian parliamentary archive;
Coding in Missing Legislatures (BW).r is the code for filling out a few legislatures in the \cite{fetscher} dataset where member and chair information were missing;
Code for Building Variables from Dataframe.rmd and Code-for-Building-Variables-from-Dataframe.pdf contain code for calculating the right-hand side variables (both the markdown file and the knitted pdf are included here);
Master Thesis Analysis.rmd and Master-Thesis-Analysis.pdf contain the code for running all regressions and other tests for this project (both the markdown file and the knitted pdf are included here)

Code by Jannick Steinacker: 
Batch_Cosine_Similarity_v6.py is the code for deriving the cosine similarity values;
batch_dictionary_delta.py contains code to calculate the dictionary deltas
